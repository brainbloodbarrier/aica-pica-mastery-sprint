## Spaced Repetition Implementation

### Micro-Level Spacing (Within Modules)

**Blocked Practice (Modules 1-8)**:
- Questions within each module focus on related concepts
- Average spacing: 1-5 questions between related concepts
- **Purpose**: Initial encoding and pattern recognition
- **Benefit**: Builds strong foundational schemas

**Interleaved Practice (Module 9)**:
- 33 questions drawn from 8 source modules (M1-M8)
- Distribution: M1:3, M2:5, M3:4, M4:5, M5:4, M6:4, M7:4, M8:4
- Questions interleaved by module using round-robin pattern
- **Purpose**: Spaced retrieval after 30-50 question interval
- **Benefit**: Strengthens long-term retention and retrieval fluency

### Macro-Level Spacing (Across Modules)

**Progressive Re-exposure Pattern**:

1. **Initial Learning** (Modules 1-5):
   - AICA: M1 → M2 (5 questions) → M3 (10 questions)
   - PICA: M1 → M4 (12 questions) → M5 (10 questions)
   - Spacing: 5-15 questions between AICA exposures

2. **Comparative Integration** (Module 6):
   - Direct AICA vs PICA comparison
   - Spacing: 30-40 questions after M5
   - **Critical spaced retrieval window**

3. **Applied Learning** (Modules 7-8):
   - Quantitative data (M7): 20 questions
   - Surgical applications (M8): 12 questions
   - Spacing: 45-55 questions after initial learning

4. **Comprehensive Interleaving** (Module 9):
   - All concepts revisited
   - Spacing: 65-80 questions after initial exposure
   - **Optimal spaced retrieval interval**

5. **Mastery Synthesis** (Module 10):
   - Complex multi-step reasoning
   - Spacing: 90-120 questions after initial learning
   - **Ultimate retention test**

## Evidence-Based Learning Principles

### 1. Progressive Complexity

**Modules 1-5**: Foundational knowledge
- Simple recall and recognition
- Basic anatomical relationships
- Single-concept questions

**Modules 6-8**: Integration and application
- Comparative analysis
- Quantitative precision
- Clinical vignettes

**Modules 9-10**: Synthesis and mastery
- Multi-concept integration
- Complex reasoning
- Expert-level decision-making

### 2. Interleaving Schedule

**Module 9 Implementation**:
```
Question sequence: M1→M2→M3→M4→M5→M6→M7→M8→M1→M2→...
```
- Prevents blocking of similar questions
- Forces retrieval from long-term memory
- Enhances discrimination between similar concepts

### 3. Retrieval Practice

**Cumulative Pattern**:
- M1 concepts tested in: M1 → M6 → M9 → M10 (4 exposures)
- Spacing: Initial → +30q → +65q → +90q
- Follows forgetting curve optimization

### 4. Difficulty Gradient

**Pass Thresholds**:
- Modules 1-6: 80% (foundational)
- Module 7, 9: 85% (precision required)
- Module 10: 90% (mastery certification)

Progressive difficulty ensures engagement without overwhelming cognitive load.
